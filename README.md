# Pre-ML

Ce dossier contient tous mes travaux préparatoires avant de passer au Machine Learning.  
L’objectif est de pratiquer l’**exploration de données** (EDA), la **visualisation** et la **préparation de features**.

## Contenu

- `loader.py` : fonctions pour charger et nettoyer les données CSV
- `stats.py` : calcul de statistiques simples (min, max, moyenne, médiane, langues par décennie)
- `visualize.py` : visualisations avec matplotlib (bar chart, histogramme, cumul)
- `data/` : fichiers CSV utilisés pour les tests
- `figures/` : graphes sauvegardés en `.png`

## Objectifs

- S’entraîner à manipuler des dictionnaires et listes
- Produire des statistiques descriptives
- Visualiser les données (histogrammes, bar charts, cumul)
- Générer des features exploitables pour du ML futur

## Usage

1. Placer les fichiers CSV dans le dossier `data/`
2. Lancer les scripts pour générer statistiques et graphiques
3. Les graphiques sont sauvegardés dans `figures/`
